## 2. Description du service

yuAs69Bots est un bot Discord communautaire **tout-en-un** developpe
en Python avec discord.py et MongoDB. Il fournit un ecosysteme complet
de fonctionnalites pour gerer une communaute :

**Moderation et securite :**
- Kick, ban, unban, timeout, clear, slowmode, lock/unlock
- Anti-raid avec detection IA basee sur un score de trust
  (age du compte, avatar par defaut, patterns de username suspects)
- Anti-spam (burst, mentions, liens) avec timeout auto
- Logs serveur complets (message edit/delete, role changes, voice, etc.)

**Engagement et roles :**
- Systeme XP avec leaderboards et level-roles automatiques
- Reaction roles (panneaux emoji -> role persistants)
- Button roles (panneaux a boutons)
- Auto-onboarding interactif (quiz aux nouveaux membres pour
  attribuer automatiquement les bons roles)
- Welcome / goodbye personnalisables avec embeds riches

**Outils communautaires :**
- Giveaways persistants avec participation par bouton
- Auto-reponses sur mots-cles configurables
- Commandes custom (prefixe `?nom`)
- Suggestions communautaires (desactivees pour l'instant)
- Newsletter / annonces opt-in en DM (`/pub send`)
- Rappels persistants (`/remind`)
- Tickets DM <-> threads bidirectionnels

**Verification anti-bot :**
- Systeme de verification gate par user-app install (anti-bot fort)
- Lockdown automatique du serveur tant que non verifie

**Administration :**
- Systeme de tier free / premium (`/redeem`, `/botadmin gencode`)
- Trial premium 3 jours offert par serveur

yuAs69Bots est concu, developpe et maintenu par **jibzz69** de maniere
independante. Le code source du bot est proprietaire et non distribue
publiquement.
